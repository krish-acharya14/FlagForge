## Ways to Contribute

You don't have to write code to contribute to FlagForge.

We welcome contributions such as:

* Bug reports
* Feature requests
* Documentation improvements
* UI/UX suggestions
* Performance improvements
* New attachment analysis tools
* New converter operations
* Tutorials and educational content
* Testing on different platforms

Every contribution, big or small, helps make FlagForge better.
