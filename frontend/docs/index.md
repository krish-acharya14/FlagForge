# Welcome to FlagForge

**FlagForge** is a free and open-source desktop application designed to simplify the process of solving and documenting **Capture The Flag (CTF)** challenges. Whether you're a beginner learning the basics of cybersecurity or an experienced player competing in major CTF competitions, FlagForge provides a centralized workspace for organizing challenges, analyzing files, experimenting with encodings and cryptography, and writing professional-quality writeups.

Unlike traditional note-taking applications, FlagForge is built specifically for CTFs. Every feature is designed around the workflow of a player: from importing challenge files to documenting the final exploit.

---

## Why FlagForge?

CTF competitions often involve dozens of challenges spread across multiple categories, each with numerous files, commands, screenshots, scripts, and notes. Managing all of this across different applications quickly becomes messy.

FlagForge combines everything into a single workspace by providing:

* Challenge management
* Integrated attachment analysis
* Built-in conversion and cryptography tools
* Extensive documentation and learning resources
* Markdown-based writeups
* Fully offline workflow
* Open-source and community driven

The goal is simple: spend less time switching between tools and more time solving challenges.

# Features

## 1. Challenge Management

Organize your CTFs into projects and create individual challenges for each problem. Every challenge contains its own writeup, attachments, notes, flag, metadata, and tags, making it easy to keep everything structured.

---

## 2. Attachment Tools

Each challenge can contain one or more attachments, including:

* Images
* PCAP files
* Executables
* Archives
* Documents
* Audio and video
* Source code
* Memory dumps
* Disk images
* And many more

FlagForge automatically detects the uploaded file type and presents a collection of relevant analysis tools. Instead of manually opening external utilities, you can quickly inspect and analyze files directly within your workflow.

Examples include:

* Viewing metadata
* Hex inspection
* Strings extraction
* Image analysis
* Archive exploration
* Binary inspection
* File hashing
* And many additional utilities depending on the attachment type.

---

## 3. Converter Tools

Inspired by CyberChef, the Converter section provides a growing collection of utilities for transforming and decoding data.

Available categories include:

### Encodings

* Base64
* Base32
* Base58
* Hex
* Binary
* URL Encoding
* ASCII

### Cryptography

* AES
* DES
* RSA
* XOR
* ROT13
* Caesar Cipher
* Vigenère Cipher
* Rail Fence
* And more

### Hashing

* MD5
* SHA-1
* SHA-256
* SHA-512
* Bcrypt
* RIPEMD160

The converter tools are designed to make common CTF operations quick and accessible without leaving the application.
